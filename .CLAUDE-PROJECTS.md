## Layer 2: Code Graph (Graphify)

**What it does:**
- Auto-generates 1000+ node graph of functions/classes/files
- Updates incrementally on every commit
- Runs in background via git hook (doesn't block commits)

**Git hook:**
```bash
#!/bin/bash
# .git/hooks/post-commit
cd "$(git rev-parse --show-toplevel)"
graphify update . > /dev/null 2>&1 &
```

**Maintenance:**
- Auto runs on every commit
- No manual intervention needed
- Output: `graphify-out/graph.json`

## Layer 3: Pattern Search (Mempalace)

**What it does:**
- Semantic search across code patterns
- Auto-indexes on every commit
- 500+ "drawers" of related code patterns

**Configuration:**
```yaml
# .mempalace/config.yaml
version: 1
project: "[PROJECT_NAME]"
auto_update: true
index_on_commit: true
exclude_patterns:
  - "node_modules/"
  - ".git/"
  - "dist/"
  - "build/"
```

**Usage in Claude Code:**
```
/mempalace query "pattern description"
```

**Maintenance:**
- Auto updates on every commit
- No manual intervention needed

## Layer 4: Tool Compression (NeuralMind)

**What it does:**
- Automatically compresses Read/Bash/Grep output before Claude sees it
- 88% savings on file reads (full file → skeleton)
- 91% savings on bash output (verbose → error summary)

**Installation:**
```bash
neuralmind install-hooks .
```

**What happens automatically:**
- Every `Read` tool call → graph skeleton (functions + call graph)
- Every `Bash` tool call → error lines + summary
- Every `Grep` call → top 25 matches + count

**No configuration needed** — just works.
