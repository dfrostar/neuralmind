# 🧠 NeuralMind

**Adaptive Neural Knowledge System for AI Code Understanding**

Achieve **40-70x token reduction** when working with AI coding assistants. NeuralMind creates intelligent, query-aware context from your codebase knowledge graphs.

## Features

- **Massive Token Reduction**: 40-70x fewer tokens than loading full codebases
- **4-Layer Progressive Disclosure**: Only load what's relevant to your query
- **Semantic Search**: Find code entities by meaning, not just keywords
- **Community Awareness**: Understands logical code clusters
- **MCP Integration**: Works with Claude, Cursor, and other MCP tools
- **Incremental Updates**: Only re-embed changed nodes

## Quick Start

### Installation

```bash
# From PyPI
pip install neuralmind

# With MCP support
pip install neuralmind[mcp]

# From source
git clone https://github.com/agent-zero/neuralmind.git
cd neuralmind
pip install -e .
```

### Requirements

- Python 3.10+
- A `graph.json` knowledge graph (generated by [graphify](https://github.com/safishamsi/graphify))

### Basic Usage

```python
from neuralmind import NeuralMind

# Initialize for your project
mind = NeuralMind('/path/to/project')
mind.build()  # Generate embeddings from graph.json

# Wake-up context (~600 tokens) for starting a conversation
wakeup = mind.wakeup()
print(wakeup.context)
print(f"Tokens: {wakeup.budget.total}")

# Query context (~1000-1500 tokens) for specific questions
result = mind.query("How does authentication work?")
print(result.context)
print(f"Token reduction: {result.reduction_ratio:.1f}x")
```

### CLI Usage

```bash
# Build neural index
neuralmind build /path/to/project

# Query the codebase
neuralmind query /path/to/project "How does authentication work?"

# Get wake-up context
neuralmind wakeup /path/to/project

# Run benchmark
neuralmind benchmark /path/to/project

# Direct semantic search
neuralmind search /path/to/project "authentication"

# Show statistics
neuralmind stats /path/to/project
```

### MCP Server

```bash
# Run the MCP server
python -m neuralmind.mcp_server

# Or if installed
neuralmind-mcp
```

Add to your Claude Desktop config:

```json
{
  "mcpServers": {
    "neuralmind": {
      "command": "neuralmind-mcp"
    }
  }
}
```

## How It Works

### 4-Layer Architecture

NeuralMind uses progressive disclosure to minimize token usage:

| Layer | Tokens | Content | When Loaded |
|-------|--------|---------|-------------|
| L0: Identity | ~100 | Project name, description | Always |
| L1: Summary | ~500 | Architecture overview, code clusters | Always |
| L2: On-Demand | ~200-500 | Relevant code modules | Per query |
| L3: Search | ~200-500 | Semantic search results | Per query |

### Token Budget

- **Wake-up**: L0 + L1 = ~600 tokens
- **Query**: L0 + L1 + L2 + L3 = ~1000-1500 tokens
- **Full codebase**: 50,000+ tokens
- **Reduction**: 40-70x typical

## Integration with Graphify

NeuralMind works best with [graphify](https://github.com/safishamsi/graphify) knowledge graphs:

```bash
# Install graphify
pip install graphifyy

# Generate knowledge graph
graphify update /path/to/project

# Build NeuralMind index
neuralmind build /path/to/project
```

## API Reference

### NeuralMind Class

```python
class NeuralMind:
    def __init__(self, project_path: str, db_path: str = None): ...
    def build(self, force: bool = False) -> Dict: ...
    def wakeup(self) -> ContextResult: ...
    def query(self, question: str) -> ContextResult: ...
    def search(self, query: str, n: int = 10, **filters) -> List[Dict]: ...
    def benchmark(self, sample_queries: List[str] = None) -> Dict: ...
    def export_context(self, query: str = None, output_path: str = None) -> str: ...
    def get_stats(self) -> Dict: ...
```

### ContextResult

```python
@dataclass
class ContextResult:
    context: str              # The optimized context text
    budget: TokenBudget       # Token usage breakdown
    layers_used: List[str]    # Which layers were loaded
    communities_loaded: List[int]  # Code clusters accessed
    search_hits: int          # Number of search results
    reduction_ratio: float    # Compared to full codebase
```
